# BDSPS QR Landing Page

Files included:
- index.html
- styles.css
- script.js
- favicon.svg

## GitHub Pages setup
1. Upload all files to the root of your `qr` repository.
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select branch **main** and folder **/root**.
5. Save.

Your page should publish at:
`https://pragalbhdwivedi.github.io/qr/`

## Notes
- The page asks for English/Hindi every time it loads.
- It uses the live BDSPS logo from `https://bdsps.in/assets/img/logo.svg`.
- App logos are loaded via Bootstrap Icons CDN.
